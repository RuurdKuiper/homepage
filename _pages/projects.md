A curated selection of public projects from GitHub and Hugging Face.

## GitHub projects

### [lad-code](https://github.com/RuurdKuiper/lad-code)
Codebase for LoRA-adapted diffusion style language modeling experiments and research workflows.

### [tini-lad](https://github.com/RuurdKuiper/tini-lad)
Compact implementation and interface around diffusion-inspired text generation concepts.

### [radiology-llm](https://github.com/RuurdKuiper/radiology-llm)
LLM-centered orchestration ideas for radiology workflows and automation prototypes.

### [auto-inspect-sr](https://github.com/RuurdKuiper/auto-inspect-sr)
Automated trustworthiness checks for randomized controlled trial reporting via LLM prompting.

## Hugging Face spaces

### [LAD Space](https://huggingface.co/spaces/Ruurd/lad)
Interactive demo space for experimenting with LAD variants and hyperparameter behavior.

### [radioLM Space](https://huggingface.co/spaces/Ruurd/radiolm)
Public space demonstrating language-model-driven approaches for radiology-oriented tasks.
